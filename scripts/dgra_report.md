## 方法学附录

### 分析背景: 通用健康筛查 / General Health Screening

- **GTEx 参考组织**: None

- **快速排除规则**: 非必需基因 + ClinVar良性/意义不明 + gnomAD常见 → Tier 3


### 分析流程

1. **转录本校正**: Ensembl REST API → canonical transcript → 本地回退

2. **假基因检测**: VAF 偏差分析识别已知假基因对

3. **gnomAD 整合**: AF>1% 常见; AF<0.1% 罕见; NOT_CAPTURED 明确标注

4. **蛋白功能域映射**: UniProt REST API → DOMAIN/REGION 特征 → 本地回退

5. **组织相关性评估**: GTEx API → median TPM → 自动分级 + 本地回退

6. **表型关联分析** (v0.7): LLM 语义匹配 → 基因已知表型 vs 用户输入表型 → 仅 Tier 1/2 执行

7. **三级分类**: Action (Tier 1) → Inform (Tier 2) → No concern (Tier 3)

8. **基因检测历史记录**: 保留变异信息用于后续分析

9. **缓存**: 所有 API 响应缓存 30 天 (SQLite); 离线模式仅用缓存
